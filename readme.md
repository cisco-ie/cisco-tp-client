> A node.js API client for Cisco Telepresence-enabled endpoints


## Install

```
$ npm install --save cisco-tp-client
```


## Usage

```js
const ciscoTpClient = require('cisco-tp-client');

ciscoTpClient('unicorns');
//=> 'unicorns & rainbows'
```
